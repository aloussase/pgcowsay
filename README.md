# pgcowsay

This is a postgres extension I made for educational purposes.

```sql
select pgcowsay.say('Hello!');
```

Output:

```
             say
------------------------------
  _______                    +
 < Hello >                   +
  -------                    +
         \   ^__^            +
          \  (oo)\_______    +
             (__)\       )\/\+
                 ||----w |   +
                 ||     ||   +

(1 row)
```
